# pickle-linux

The Pickle Linux repositories.

[Check here for installation instructions.](https://github.com/Stilic/pkh?tab=readme-ov-file#bootstraping-pickle-linux)

## `main`

This is the repository holding all the essential packages needed for the distribution to function correctly.
There isn't any dependency specified in those packages as they're all always installed together.

## `user`

This is the repository for all optional packages that can be installed on top of the base system.
